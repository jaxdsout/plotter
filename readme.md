# Atlas (fka Plotter)

Full CRUD web app with mapmaking features, client management, invoice logging & tracking, and more. All data is stored on custom RESTful API which can be found below.

[Django backend](https://github.com/jaxdsout/plotter_backend)

### Frameworks & Packages

React via CRA, Redux, ChartJS, Mapbox, BeautifulDND, Semantic UI, Tailwind
